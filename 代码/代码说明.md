# 代码包说明

`vm_source/` 于 2026-07-21 通过 SSH 从虚拟机 `~/unitree_h1_learn` 按目录同步。该目录含 **50 个文件（源码、配置、XML 和运行说明）**，不是只有最初列出的 4 个验证脚本；所有文件哈希均记录在根目录 `CHECKSUMS.sha256`。

| 目录 | 内容 | 作用 |
| --- | --- | --- |
| `ACT/` | ACT 训练、推理、DETR/VAE 网络、工具函数及两份配置 | 红棒传递数据集的策略训练和评估。 |
| `DataCollecter/` | H1 采集、动作策略、HDF5 检查 | 演示数据采集与 episode 结构验证。 |
| `Mujoco_env/` | H1 MuJoCo 环境、IK、KDL 工具、机器人封装和必要 XML 模型 | H1 仿真、运动学与逆运动学。 |
| `validation/` | H1 专家轨迹录制、ACT/HDF5 校验、G1 舞蹈和策略切换证据脚本 | 生成视频、关键帧和可复核日志。 |
| `project_execution/scripts/` | 两个训练监控脚本 | 12000 轮和中等规模训练过程监控。 |
| 根目录 | `README.md`、`.gitignore` | 原虚拟机工作区的运行说明和忽略规则。 |

重点适配/验证脚本为：

- `validation/record_robojudo_g1_loco_dance_switch.py`：G1 行走、舞蹈、回到行走的视频记录与事件日志；
- `validation/run_robojudo_beyondmimic_evidence.py`：固定仿真步数的 RoboJuDo 证据导出；
- `DataCollecter/analyze_hdf5.py`：HDF5 episode 结构检查；
- `ACT/config.cpu.yaml`：ACT 的 CPU 降级训练配置。

为控制仓库体积并保持依赖边界，本代码包不包含 HDF5 原始数据、训练检查点、缓存、完整 GR00T 克隆、RoboJuDo 源码、Unitree SDK 或其模型资源。RoboJuDo 是外部依赖，见 [HansZ8/RoboJuDo](https://github.com/HansZ8/RoboJuDo)；复现时需自行获取并遵守其许可证与依赖说明。

源文件仅保留一份于 `vm_source/`，不再提供内容重复的 ZIP/TAR.GZ 压缩包。
